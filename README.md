# NFCify

macOS에서 파일명의 유니코드 정규화(NFD → NFC 변환)를 자동으로 처리하는 앱

macOS는 파일명을 NFD(Normalization Form Decomposed) 형식으로 저장하여 한글, 프랑스어, 독일어 등의 특수문자가 분리되는 문제가 발생합니다. NFCify는 이를 NFC(Normalization Form Composed) 형식으로 자동 변환하여 문자 분리 문제를 해결합니다.

**지원 언어:** 한글, 프랑스어, 독일어, 스페인어 등 모든 유니코드 문자

## 📥 다운로드

최신 버전은 [Releases 페이지](https://github.com/nfcify/nfcify/releases/latest)에서 다운로드할 수 있습니다.

### 설치 방법

1. [Releases 페이지](https://github.com/nfcify/nfcify/releases/latest)에서 최신 버전의 `NFCify.dmg` 다운로드
2. DMG 파일을 열고 `NFCify.app`을 Applications 폴더로 드래그
3. NFCify 실행

## ✨ 주요 기능

- **자동 파일명 변환**: macOS의 NFD 형식 파일명을 NFC 형식으로 자동 변환
- **실시간 모니터링**: 파일 생성/변경을 실시간으로 감지하여 즉시 변환
- **로그인 시 자동 실행**: 시스템 시작 시 자동으로 백그라운드에서 실행
- **다국어 지원**: 한국어, 영어 인터페이스 지원
- **메뉴바 통합**: 간편한 메뉴바 아이콘으로 빠른 접근 및 제어

## 🔧 시스템 요구사항

- macOS 10.15 (Catalina) 이상
- Python 3.10 이상 (앱에 포함되어 있어 별도 설치 불필요)

## 📖 사용 방법

### 1. 앱 실행

Applications 폴더에서 `NFCify` 앱을 실행하면 메뉴바에 아이콘이 표시됩니다.

### 2. 로그인 시 자동 실행 설정

메뉴바 아이콘 클릭 → **로그인 시 자동 실행** 체크

### 3. 언어 설정

메뉴바 아이콘 클릭 → **언어 / Language** → 원하는 언어 선택 (한국어 / English)

선택한 언어는 자동으로 저장되며 다음 실행 시에도 유지됩니다.

### 4. 앱 종료

메뉴바 아이콘 클릭 → **종료**

## 🔄 업데이트

NFCify는 자동으로 업데이트를 확인합니다. 새 버전이 있으면 앱 내에서 알림을 받게 됩니다.

수동으로 업데이트를 확인하려면:
- 메뉴바 아이콘 클릭 → **업데이트 확인**

## 🐛 문제 해결

### 파일명이 변환되지 않아요

1. NFCify가 실행 중인지 확인 (메뉴바에 아이콘이 있어야 함)
2. 파일이 모니터링 대상 디렉토리에 있는지 확인
3. 앱을 재시작해보세요

### 로그인 시 자동 실행이 작동하지 않아요

1. 메뉴바에서 **로그인 시 자동 실행** 옵션이 체크되어 있는지 확인
2. 시스템 환경설정 → 사용자 및 그룹 → 로그인 항목에서 NFCify가 등록되어 있는지 확인
